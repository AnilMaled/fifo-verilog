# Synchronous FIFO Design in Verilog

## Overview
This project implements an 8-bit synchronous FIFO using Verilog HDL.

The FIFO supports:
- Read operation
- Write operation
- Full flag detection
- Empty flag detection
- Circular buffer implementation
